# Serviceheft v2

Ein digitales Serviceheft zur strukturierten Dokumentation von Wartungen, Reparaturen und Fahrzeughistorien.

## Projektstruktur

- **frontend/**: Enthält die Benutzeroberfläche (React/TypeScript).
- **backend/**: Enthält die API und Geschäftslogik (Node.js/Python).
